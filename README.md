## 多线程
- **Q1：用多线程的目的？**

 &emsp; 快，充分利用cpu，并发做多件事

- **Q2：单核CPU市适合用多线程吗？**

 &emsp; 适合，若是单线程，线程中需要等待IO操作时，CPU就空闲出来了。

- **Q3：线程什么时候会让出CPU?**

 &emsp; 阻塞时：wait，await, 线程中需要等待IO操作时

 &emsp; sleep时

 &emsp; yied时

 &emsp; 线程结束时
